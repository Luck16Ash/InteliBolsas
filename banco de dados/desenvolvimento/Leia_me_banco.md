### ARQUIVOS DO BANCO DE DADOS INTELIBOLSAS

**1. intelibolsas_banco_parte1**
* Cria o banco de dados 'intelibolsas'.
* Cria todas as tabelas, chaves primarias e estrangeiras.
* Insere o usuario administrador padrao.
* Define relacionamentos e integridade referencial.

**2. intelibolsas_banco_parte2**
* Adiciona colunas utilitarias (ativo, updated\_at).
* Cria procedures de cadastro, atualizacao, inscricao, favoritos e suspensao.
* Cria view de relatorio de inscricoes por curso.
* Responsavel pela logica do CRUD e manutencao.

---

**Observacao:** Ambos os scripts foram testados no phpMyAdmin (MariaDB/MySQL via XAMPP).
