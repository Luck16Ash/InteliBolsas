# Projeto CRUD - InteliBolsas

Este projeto é um sistema CRUD desenvolvido em PHP com PDO e MariaDB, de modo que facilite a sua manutenção e escalabilidade.

## Estrutura de Pastas
- **publico/** → Arquivos acessíveis pelo navegador (HTML, PHP, CSS, JS, imagens).
- **codigo/** → Código PHP (configurações, conexão e funções).
- **banco/** → Scripts SQL para criar e gerenciar o banco de dados.
- **documentos/** → Documentação (DER, dicionário de dados, manual de uso).
- **testes/** → Testes automatizados e modelos.

## Modo de uso
1. Importe o banco de dados com o arquivo `banco/banco_init.sql`.
2. Configure a conexão em `codigo/conexao.php`.
3. Acesse `publico/index.php` no navegador.
